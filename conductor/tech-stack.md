# Tech Stack

## Lenguaje y Framework

- **Java 21** — LTS reciente, features modernos.
- **Spring Boot 3.4+** — framework principal.
- **Maven 3.8+** — build tool.

## Cliente HTTP

- **RestClient** — cliente HTTP síncrono moderno de Spring (reemplaza RestTemplate).

## Documentación OpenAPI

- **springdoc-openapi v2** — genera y sirve la spec OpenAPI 3.0.
- **SwaggerUI** — interfaz visual para probar los endpoints.

## Testing

- **JUnit 5** — framework de tests unitarios.
- **Mockito** — mocking para servicios.
- **Selenium WebDriver** — tests de UI sobre SwaggerUI.

## Configuración

- **application.properties / application.yml** — configuración externalizada.
- **Variables de entorno** — para el Bearer Token.
